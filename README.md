# gingko

gingko's blog theme

## Installation

Add this line to your Jekyll site's `Gemfile`:

```ruby
gem "gingko"
```

And add this line to your Jekyll site's `_config.yml`:

```yaml
theme: gingko
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install gingko
